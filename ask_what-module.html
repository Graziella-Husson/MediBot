Module ask_what

source code

This module is used to regroup all ask what actions.

An ask what action is an action where the bot will ask with buttons what's wrong with the infos it has collected

get_buttons_simple(buttons, slot_name, tracker, language, intent_name, other_value=None, button_name=None)

source code 
Parameters:
  • buttons - list of buttons in which we have to append new ones
  • slot_name - name of the slot we want to add a button for
  • tracker - used to get all infos for 'slot_name' entity from the tracker
  • language - language of the bot (for display the right name of button)
  • intent_name - name of the intent linked to the slot
  • other_value - unrequired, if the slot value is set to this value, do not add a button
  • button_name - unrequired, name of the button if different to {slot_name}_button

    Add a button to the list of buttons to display, named after the slot name. If the button is clicked, set the slot value to None

Returns:
list of buttons in which we appended new ones

get_buttons_multiple(buttons, slot_names, tracker, language, intent_name, to_check)

source code 
Parameters:
  • buttons - list of buttons in which we have to append new ones
  • slot_names - list of names of the slots we want to add a button for
  • tracker - used to get all infos for 'slot_name' entity from the tracker
  • language - language of the bot (for display the right name of button)
  • intent_name - name of the intent linked to the slot
  • to_check - value to check, if not set, do not display the buttons

    Add multiple buttons to the list of buttons to display, named after the slot names. If the button is clicked, set the slot value to None If the button clicked is not named after to_check, set the slot value to "Incorrect"

Returns:
list of buttons in which we appended new ones

get_buttons_boolean(buttons, slot_name, tracker, language, intent_name, button_name=None)

source code 
Parameters:
  • buttons - list of buttons in which we have to append new ones
  • slot_name - name of the boolean slot we want to add a button for
  • tracker - used to get all infos for 'slot_name' entity from the tracker
  • language - language of the bot (for display the right name of button)
  • intent_name - name of the intent linked to the slot
  • button_name - unrequired, name of the button if different to {slot_name}_button

    Add a button to the list of buttons to display, named after the slot names. If the slot value is True, when the button is clicked, set the slot value to False If the slot value is False, when the button is clicked, set the slot value to True

Returns:
list of buttons in which we appended new ones
