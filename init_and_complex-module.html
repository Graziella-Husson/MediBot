duckling_set_slots(data, to_return, language)

source code 
Parameters:
  • data - all entities detected
  • to_return - list of event to return after action
  • language - language to use
Returns:
List of SlotSet

Check all the complex entities detected with duckling to know what slot to set:

  • time
  • distance
  • duration
  • temperature

If distance, duration or temperature have a unit, set them For every times founded, check if it's the traduction of others entity and, if not set time If time has a 'to' and 'from' date, check the duration and set it.

check(to_return, intent, entities, tracker, dispatcher, response, domain)

source code 
Parameters:
  • to_return - the list of SlotSet
  • intent - intent detected
  • entities - entities detected
  • tracker - tracker used at this time
  • dispatcher - dispatcher to use at this time
  • response - response of the user
  • domain - domain to use
Returns:
List of events to return at the end of action

Check the response of the user :

  • if the confidence of intent detection is under 50%, call Fallback action
  • if one of specific word is detected:
    • emergency: send an alert
    • stopword: tell the bot that he made a mistake
    • exitword: do not send the reminder_user_little reminder
  • if none of the above is detected :
    • Send user_reminder_little and user_reminder
