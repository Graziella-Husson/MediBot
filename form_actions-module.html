get_response_simple(response, slot_name, tracker, language, other_value=None, utt_name=None)

source code 
Parameters:
  • response - string of the response the bot will say to the user
  • slot_name - name of the slot we want to sum up
  • tracker - used to get all infos for 'slot_name' entity from the tracker
  • language - language of the bot (for display the right name of button)
  • other_value - unrequired, if the slot value is set to this value, do not sum up it
  • utt_name - unrequired, name of the ressource in json if different to slot_name Add part in the response of the bot with a sentence to sum up a given slot

    'drug' is special: sum up all infos from file FDA_druglist_2018.csv

    'time' is also special: use 'from' and 'to' infos to use the right ressource sentence.

Returns:
string of the response the bot will say to the user
