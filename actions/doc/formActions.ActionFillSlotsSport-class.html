sum_up_setted_slots(self, response, tracker, language)

source code 

The entities linked to the 'activity' intent are:

  • activity_level
  • activity_duration
  • sport
  • activity_period
  • activity_distance
  • activity_hard (boolean)
  • activity_time

The sport level is calculated with the method get_physical_activity_level(sport)

If we want the sport level, we have to have the sport slot mandatory.

Returns:
all infos sum up response
